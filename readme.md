# Proyecto 404 – Frontend

## 🚀 Cómo correr el proyecto (Windows)

### ▶ Opción rápida (Descarga inicial)
1. Haz doble clic en `start.bat`
3. Abre el navegador en `http://localhost:xxxx/`

### ▶ Correrlo (recomendada)
1. Haz doble clic en `run-dev.bat`
3. Abre el navegador en `http://localhost:xxxx/`

### ▶ Opción manual
1. Abre terminal en la carpeta del proyecto
2. Ejecuta:
```

npm install
npm run dev

```

---

## 📁 Archivos útiles

- `start.bat`: instala dependencias y ejecuta el proyecto
- `run-dev.bat`: solo ejecuta el proyecto (cuando ya está instalado)

---

## 🛠️ Requisitos

- Node.js v18 o superior
- npm instalado
