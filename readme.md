Proyecto:
	- Tema: Sistema gestor para un restaurante.
	- Nombre: RestauMaster
	- Persona objetivo: Administrador del restaurante.

	-Modulos:
		- Modulo 1: CRUD GENERAL (CRUD)
		- Modulo 2: Gestion de Tareas (Uso de calendario para llenar huecos semanales de tareas y compartirlos)
		- Modulo 3: Análisis de productos (Que productos son los más vendidos, menos vendidos, etc)
		- Modulo 4:	Desglose de ventas (Detalle con base al calendario, la tendencia de ventas).
		- Modulo 5:	Reportes (Resumen manipulable entre diferentes módulos).
